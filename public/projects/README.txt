Drop optional project thumbnails here (16:9 works best). Filenames the site
looks for:

  cryptofacial.jpg   →  CryptoFacial Recognition System
  freertos.jpg       →  FreeRTOS-Based Real-Time Robot Control
  ids.jpg            →  Intelligent Intrusion Detection for CPS
  hexaswift.jpg      →  HexaSwift Autonomous Hexapod Robot

If a file is missing, the site automatically shows a colored placeholder with
the project's initials — so this is entirely optional.

To change filenames, edit the `image:` fields in  src/data.js
